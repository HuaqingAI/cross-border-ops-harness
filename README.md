# Cross-Border Ops Harness

跨境运营 AI Harness -- 基于 BMAD 方法论的跨境电商智能运营平台

## 项目简介

本项目旨在构建一个 AI 驱动的跨境电商运营平台，整合产品管理、供应链协调、多渠道销售、合规管理等核心业务流程，通过智能化手段提升跨境运营效率。

## 技术栈

- **方法论**: BMAD (Business Method for Agile Development)
- **开发流程**: CIS (Continuous Integration Stories)
